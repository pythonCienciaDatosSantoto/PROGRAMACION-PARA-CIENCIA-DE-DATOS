üìä DataFrames y Exploraci√≥n de Datos#

Si la Serie es la columna, el DataFrame es la tabla completa. Esta es la estructura de datos m√°s importante y utilizada en Pandas. En este notebook, aprender√°s a dar tus primeros pasos como un explorador de datos: cargar un conjunto de datos y realizar una inspecci√≥n inicial para entender con qu√© est√°s trabajando.

  • 2.1. Cargando Datos desde un Archivo CSV

La mayor√≠a de las veces, tus datos estar√°n en archivos. Uno de los formatos m√°s comunes es el CSV (Comma-Separated Values). Pandas hace que cargar estos archivos sea incre√≠blemente f√°cil con pd.read_csv().

import pandas as pd

# Usaremos un dataset muy popular y amigable para empezar: "Palmer Penguins".
# Contiene mediciones de diferentes especies de ping√ºinos.
url = 'https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv'

# Cargamos los datos desde la URL en un DataFrame llamado 'df'
df = pd.read_csv(url)

print("¬°Dataset cargado exitosamente!")

¬°Dataset cargado exitosamente!

Par√°metros √∫tiles en pd.read_csv():

  • sep: Especifica el separador de columnas. Por defecto es ,, pero a veces puede ser ; o \t (tabulador).

  • header: Indica qu√© fila usar como encabezado (nombres de columna).

  • index_col: Permite designar una columna como el √≠ndice del DataFrame.

  • nrows: √ötil para cargar solo una porci√≥n del archivo, ideal para datasets muy grandes.

Selecci√≥n por Etiqueta (.loc) y Posici√≥n (.iloc)#

Esta es la forma m√°s poderosa y correcta de seleccionar filas y columnas.

  • df.loc[...]: Selecciona por nombre de √≠ndice y columna (etiquetas).

  • df.iloc[...]: Selecciona por posici√≥n num√©rica (√≠ndices enteros).

# --- Usando .loc (por etiqueta) ---

# Seleccionar la fila con la etiqueta de √≠ndice 3
print("Fila con √≠ndice 3:")
display(df.loc[3])

# Seleccionar las filas de la 3 a la 6 y las columnas desde 'species' hasta 'sex'
print("\nSlice de filas y columnas con .loc:")
display(df.loc[3:6, 'species':'sex'])


# --- Usando .iloc (por posici√≥n) ---

# Seleccionar la cuarta fila (posici√≥n 3)
print("\nCuarta fila (posici√≥n 3):")
display(df.iloc[3])

# Seleccionar las primeras 5 filas y las primeras 3 columnas
print("\nSlice de filas y columnas con .iloc:")
display(df.iloc[0:5, 0:3]) # Nota: el final del slice (5 y 3) no se incluye

Bonus: Acceso R√°pido a un Solo Valor con .at e .iat#

Si necesitas obtener o modificar un √∫nico valor, .at (por etiqueta) e .iat (por posici√≥n) son mucho m√°s r√°pidos que .loc e .iloc.
